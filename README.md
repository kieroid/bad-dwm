suckless dwm
============================
simple fork of dwm-6.4 with the patches that i used pre-applied.

patches
------------
* dwm-combo
* dwm-alpha
* dwm-cfacts-vanitygaps
* dwm-restartsig
* dwm-centretitle
* dwm-exitmenu

installation
------------
edit config.mk to match your system<br/>
type `doas make install` to build dmenu, then install.
